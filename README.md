# Ocean-Support

通过 [Ocean](https://github.com/zhangz1w3nCode/ocean) 平台构建的原子能力集合。

## 为什么需要 Ocean-Support

Ocean 的 Agentic 构建会参考当前项目下已有的原子能力作为样本——样本质量直接决定生成质量。

但大多数用户刚接触 Ocean 时，手里并没有好样本可以参考。

Ocean-Support 就是用来解决这个问题的：

- **新手参考** —— 为刚接触 Ocean 的用户提供可直接借鉴的能力范例，降低构建门槛。
- **Agentic 样本源** —— 将经过验证的高质量能力沉淀于此，让 Agentic 构建有好的参考，从而生成更好的内容。
